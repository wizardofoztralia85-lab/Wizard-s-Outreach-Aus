# Wizard's Outreach Australia

Central hub for Glass Wizard Australia to spread our message and get down to business.

## 🧙 Admin Dashboard

We've created a powerful **Admin Front-End Updater** to help manage the Wizard's Outreach ecosystem!

### Quick Start

1. Open the admin dashboard:
   ```bash
   open admin/index.html
   ```
   Or visit it in your browser at: `file:///path/to/admin/index.html`

2. The dashboard will automatically:
   - Refresh every 30 seconds
   - Monitor repository status
   - Display recent activity
   - Show system health

### Features

- ✅ **Auto-refresh** - Updates every 30 seconds automatically
- 📊 **Repository monitoring** - Track Lyric Live and Subscription Starter repos
- 📰 **Activity feed** - See recent commits, issues, and PRs
- ⚡ **Quick actions** - One-click access to repositories
- 🏥 **Health monitoring** - Check API and Firebase status
