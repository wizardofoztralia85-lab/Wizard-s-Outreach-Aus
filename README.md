# Wizard-s-Outreach-Aus
Admin-facing front end for keeping Wizard outreach content current.

## Projects
- `admin-updater/`: React + Vite single-page app that lets admins update hero copy, CTA links, announcements, and targeted content blocks with a live preview. Changes persist locally so you can draft safely before publishing.

## Quick start
```bash
cd admin-updater
npm install
npm run dev
```

## Ship it
- Run `npm run build` inside `admin-updater` to ensure the UI compiles before handing off or deploying.
